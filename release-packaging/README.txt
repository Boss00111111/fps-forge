INSTALL (EN)
------------
1. Extract this ZIP into any folder. Keep ALL files together (do not move only
   the .exe away from the rest).
2. Double-click "FPS Forge.exe".
3. Enter your Premium license key from your purchase (Gumroad / email).
   This build already includes the license server URL — you only enter the key.

If Windows SmartScreen says "Windows protected your PC":
  • Click "More info" → "Run anyway", OR
  • Right-click "FPS Forge.exe" → Properties → check "Unblock" → Apply → OK.

WHAT THE APP SHOWS
------------------
RAM, CPU, disk and process counts are read from your real Windows system — not
decorative fake numbers.

ONE-CLICK BOOST (Premium)
-------------------------
Applies High Performance power plan, cleans temp files, flushes DNS, closes
tracked background programs (launchers / updaters / some meeting apps — not
your game by design), then asks Windows to trim process working sets. It does
not remove RAM that browsers or IDEs are actively using; close heavy apps for a
big drop in RAM %.
